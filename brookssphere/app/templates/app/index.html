What Does This Site Do?

This site has no "real" functionality. When you login, you'll have the opportunity to save, retrieve, edit, and delete essentially random pieces of information to the database. The only purpose is to show beginners what's happening behind the scenes.

This site is built with Python (Django Rest Framework) for the backend, JavaScript (React framework) for the frontend (with lots of pre-generated CSS), and a PostgreSQL Database.

Different Levels

For each piece of information, you'll see 2 levels of detail.

Level 1 is the simple stuff, perfect for people who are learning the foundations of full-stack web development.

Level 2 goes a bit more in-depth. If you already have the basics down, this will drill into how functionality is implemented.

What Happened When You Landed On This Home Page

Example: When you landed on this site Django resolved the URL with this:

        urlpatterns = [
        path('', views.home, name='home'),
        ...

That empty quote at the start meaning the home page with nothing after - just brksmtchll.

The views.home called the function below that renders the HTML file at app/index.html

            def home(request):
    """Renders the home page."""
    assert isinstance(request, HttpRequest)
    return render(
        request,
        'app/index.html',
        {
            'title':'Home Page',
            'year':datetime.now().year,
            'meta_info': get_meta_string(request)
        }
    )

That 'meta_info' calls another function to see get meta-data from the system that requested the site:
